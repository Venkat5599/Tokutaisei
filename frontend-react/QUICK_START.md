# ⚡ Quick Start - 3 Steps

## 1️⃣ Install

```bash
cd frontend-react
npm install
```

## 2️⃣ Configure

```bash
cp .env.example .env
# Edit .env with your values
```

## 3️⃣ Run

```bash
npm run dev
```

Open **http://localhost:3000** 🎉
